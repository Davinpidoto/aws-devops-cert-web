AWS Tagging

Amazon Web Services (AWS) allows customers to assign metadata to their AWS resources in the form of tags. Each tag is a simple label consisting of a customer-defined key and an optional value that can make it easier to manage, search for, and filter resources.

Tags

  • Key value pairs
  • Values can have multiple values
  • Tags enable you to organise resources and track your costs
  • Tags can be used with IAM to restrict resources
  • Cost allocation tags need to be enabled in the billing console
  • Once enabled AWS will generate a cost application report based on the tags
  • Cost allocation tags can be aws generated ot user generated
