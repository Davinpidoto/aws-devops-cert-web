AWS Service Catalog Documentation

AWS Service Catalog allows organizations to create and manage catalogs of IT services that are approved for use on AWS.

  • Enables organisations to create and manage catalogs of products that are approved for use on AWS
  • It has an API that provides programmatic control over all end user actions as an alternative to using the AWS console
  • It allows you to create your own custom interfaces and apps
  • It allows administrators to create and distribute application stacks called products
  • Products can be grouped into folders called portfolios
  • Users can then launch and manage products themselves without requiring access to the AWS console
  • Users only see the products they are supposed to see
