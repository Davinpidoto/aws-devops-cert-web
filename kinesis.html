AWS Kinesis Documentation

Amazon Kinesis makes it easy to collect, process, and analyze real-time, streaming data so you can get timely insights and react quickly to new information.

Services

  • Kinesis Data Analytics
  • Kinesis Data Firehose
  • Kinesis Data Streams
  • Kinesis Video Streams

Kinesis Data Analytics

  • Analyse streaming data
  • Respond in real time
  • Query using SQL
  • No servers required

Kinesis Data Firehose

  • Delivers streaming data to another destination
  • No applications to manage
  • Just configure the producer
  • Data can be transformed before delivery
Destinations
  • S3
  • Redshift
  • Elastic Search
  • Splunk

Kinesis Data Streams

  • Collects streaming data
  • Can capture gigabytes per second
  • Data is stored for 7 days

Kinesis Video Streams

  • Collects streaming video
  • Enables live and on demand playback
