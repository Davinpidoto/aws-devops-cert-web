AWS EFS Documentation

Amazon Elastic File System (Amazon EFS) provides a simple, scalable, fully managed elastic NFS file system for use with AWS Cloud services and on-premises resources.

EFS

  • Encryption at rest and in transit
  • Can be mounted to on prem servers via Direct Connect or AWS VPN

Performance Modes

General Purpose
  • Default
  • Low latency
  • Suitable for websites and home directories
Max IO
  • Higher latency
  • Suitable for Big Data

Storage Classes

Standard
  • Default
  • Frequent Access
EFS IA
  • Infrequent Access
  • Cheaper
