AWS Inspector Documentation

Amazon Inspector is an automated security assessment service that helps you test the network accessibility of your Amazon EC2 instances and the security state of your applications running on the instances.

  • Network Assessments checks for ports reachable outside of the VPC. There is an optional agent to install on EC2 which finds processes that are reachable on a given port
  • Host Assessments requires an agent to be installed on each instance
  • A report can be generated and downloaded
