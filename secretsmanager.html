AWS Secrets Manager Documentation

AWS Secrets Manager helps you protect secrets needed to access your applications, services, and IT resources. The service enables you to easily rotate, manage, and retrieve database credentials, API keys, and other secrets throughout their lifecycle.

Features

  • Encrypts secrets at rest using encryption keys from KMS
  • Secrets can be database credentials, passwords, third party api keys or even text
  • Storage and access to the secretes are available from secrets manager console, cli, api or sdk
  • Hardcoded credentials are replaced with an API call
  • Secrets can be rotated automatically according to a given schedule
